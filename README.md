# haruntuncelli.com

Personal portfolio site for Harun Tuncelli — Director of Product at Lycia AI.

Static HTML/CSS, no build step. Designed to be deployed to any static host (Vercel, Netlify, GitHub Pages, Cloudflare Pages).

## Structure

- `index.html` — homepage / portfolio cover
- `portfolio.css` — design system (purple palette, custom layout)
- `decks/` — strategy PDFs and case studies linked from the homepage

## Local preview

```bash
cd ~/Desktop/haruntuncelli.com
python3 -m http.server 8000
# then open http://localhost:8000
```
